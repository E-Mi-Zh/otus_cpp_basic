# Домашнее задание №4 «Физический симулятор»

Цель: познакомиться с классами и методами.

Задачи:

* дописать реализацию класса шара Ball (конструктор, методы, отрисовка);
* дописать конструктор класса World;
* улучшить загрузку модели из файла;
* реализовать отключение коллизии для заданных шаров;
* добавить эффект разлетающихся осколков при столкновении шаров.

Симулятор реализует упрощенную физическую модель, в которой с постоянной
скоростью движутся шары разного размера, отталкиваясь при столкновении друг с
другом или с границей области. Физический движок уже написан, в его устройстве
разбираться не требуется.
