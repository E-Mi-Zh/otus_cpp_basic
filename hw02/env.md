# Локальное окружение для сборки

## Операционная система

Рабочая ОС: Ubuntu 25.10.

## Компилятор

Для сборки использую GCC 15.2.0 и CMake 3.31.6.

## IDE

Написание кода и документации, отладка - осуществляются в IDE VS Code 1.116.0.

## Запланированный апгрейд

Ориентировочно в конце апреля - начале мая дистрибутив ОС будет обновлён на следующую
версию - 26.04 LTS. Соответственно обновятся и остальные инструменты (кроме IDE,
у неё свой график релизов). Предполагаемые версии: GCC 15.2.0, CMake 4.2.3.
